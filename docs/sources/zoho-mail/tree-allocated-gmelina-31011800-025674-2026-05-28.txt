Expected dashboard use:

- Use it as evidence that the tree allocation step happened.
- Do not assume that a customer-facing thank-you email or certificate was also sent unless there is separate evidence.
- If a matching purchase or subscription exists without this evidence, the related dashboard item should remain orange.
- If this evidence exists but no matching purchase/subscription can be found, flag as needs verification.

Suggested dashboard rule:
- Green: tree allocation email is matched with the relevant customer, invoice, transaction or order.
- Orange: allocation email exists but cannot be matched to a purchase/subscription, or related outbound customer email is missing.
- Red: tree allocation failed, has invalid coordinates, missing tree code, missing user ID, or conflicting customer data.
